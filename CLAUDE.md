# Altius Lexia — Legal Tech Firm Website

## Project
Client demo website for **Altius Lexia**, a legal tech firm in Ecuador specializing in Data Privacy, Cybersecurity, and Intellectual Property for tech companies.

## Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS 4
- **Language:** TypeScript
- **Deployment:** Vercel

## Design Guidelines
See `docs/client-guidelines.md` for full strategy, copy, and visual identity.

### Key Design Tokens
- **Primary:** Azul medianoche `#0A192F`
- **Secondary:** Gris Gunmetal `#2D3436`
- **Accent:** Verde Neon or Oro Soft
- **Typography:** Inter (sans-serif)
- **Tone:** Direct, expert, analytical, modern. Avoid dense legal jargon.
- **Inspiration:** Hogan Lovells & Fish & Richardson websites

### Content Language
All user-facing content in **Spanish** (Latin American). Code and docs in English.

## Skills
